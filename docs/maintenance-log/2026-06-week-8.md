# Week 8 Maintenance Log

This log tracks the first Phase B management-platform slice for `agentflow-oss`.

## Activities Summary

### Streaming Checkpoint Foundation

We added a phase-level checkpoint stream for Quality Loop outputs.

1. **Checkpoint Writer (`src/workflow/streaming-checkpoint.ts`)**:
   - Added append-only `streaming-checkpoints.jsonl` records.
   - Captures sprint id, step, optional iteration, phase, attempt, provider, score, tokens, cost, duration, review artifact path, output preview, and output SHA-256.
2. **Sprint Engine Integration (`src/workflow/sprint-engine.ts`)**:
   - Writes a checkpoint whenever a Quality Loop phase writes its review artifact and emits its `phase` event.
   - Preserves existing `events.jsonl` behavior as the state-transition source of truth.
3. **Offline Validation (`tests/poc-streaming-checkpoint.ts`)**:
   - Verifies produce and review checkpoints are written during a mock sprint.
   - Validates artifact paths, token counts, score, preview text, and hash stability.
